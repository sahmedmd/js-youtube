# js-youtube

Small JavaScript tools for exploring YouTube data surfaces directly from the browser console.

## Scripts

### youtube-search-velocity-scanner.js
Scans YouTube search results and estimates which videos are gaining views the fastest.

Features:
- DOM scraping of search results
- view velocity estimation
- keyword highlighting
- video ID extraction

Usage:
1. Open a YouTube search page
2. Open the browser console
3. Paste the script
4. Run it to see ranked results
